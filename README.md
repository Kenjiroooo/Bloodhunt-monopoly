# 🦟 Mosquito Bloodhunt

A custom online multiplayer board game inspired by Monopoly, reimagined with a dark mosquito and virus-themed universe. Players compete to dominate territories, collect resources, and outlast opponents in a real-time strategic battle.

---

## 🎮 Game Overview

Mosquito Bloodhunt transforms the classic Monopoly experience into a unique digital game where players take the role of mosquitoes and virus entities. The goal is to control properties, manage resources, and bankrupt all other players.

The game features real-time multiplayer functionality, interactive UI, and dynamic gameplay elements.

---

## ⚙️ Features

- 🌐 Real-time multiplayer gameplay
- 🎲 Dice rolling system with automated movement
- 🏘️ Property ownership, rent, and upgrades
- 💰 In-game currency system (Plasma)
- 🃏 Event cards (Predator Alert & Swarm Intel)
- 🔄 Trading system between players
- 💬 Live chat system
- ⏱️ Turn-based mechanics with optional timer
- 🔐 Secure game state handling (no manual tampering)

---

## 🧩 Game Mechanics

### 🏁 Starting the Game
- Each player starts with **$1500 Plasma**
- All players begin at **GO**
- Turn order is randomized

---

### 🎲 Turns
1. Roll the dice
2. Move across the board
3. Land on a tile and resolve action
4. End turn or roll again (if doubles)

---

### 🏘️ Properties
- Buy unowned properties
- Pay rent when landing on owned properties
- Upgrade properties using **blood dens (houses)** and **hives (hotels)**

---

### 🧠 Special Tiles
- **GO** → Collect Plasma
- **Quarantine** → Jail system
- **Predator Alert** → Positive or bonus effects
- **Swarm Intel** → Movement-based or risky effects
- **Taxes** → Pay penalties

---

### 🏆 Winning Condition
Be the **last player remaining** after all others go bankrupt.
